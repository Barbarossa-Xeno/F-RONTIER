# F-RONTIER

リズムゲーム「F-RONTIER」のリポジトリです。Unityで制作しています。

- 開発期間: 7か月 (2023/3 ~ 10)
- 開発環境: Unity 2021.3.16f1 → 2022.3.6f1 → 6000.3.5f2
- プラットフォーム: Android、Windows<br>
  Windows は入力のキーボード割り当てを行っていないためマウス操作となります。同時押しのシチュエーションには対応しておりません。<br>
  実際のプレイにはAndroidデバイスを推奨します。

## ゲームについて

音ゲーに触発され、

- 家でもアーケードみたいな音ゲーをプレイしたい
- いろいろな楽曲を音ゲーに仕立て上げたい

と思うようになり、このゲームの製作にチャレンジしました。

### 注意点

- 使用のしやすさの観点から、名前空間とディレクトリ構造は必ずしも対応しません。
- ほかの人のスクリプトを参照させていただいた部分には、適宜URLを貼付しています。

### 外部アセット
